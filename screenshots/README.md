## How to Add Screenshots

1. Take screenshots of each feature
2. Save them in this directory
3. Reference them in the main README.md like this:

```markdown
![Dashboard](screenshots/dashboard.png)
```

## Tips for Good Screenshots

- Use a clean, populated database (not empty)
- Show actual data and functionality
- Capture full browser window or relevant sections
- Use high resolution images
- Consider adding annotations to highlight features
